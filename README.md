## 🌱 TAPE Work

> *Tool for Agroecology Performance Evaluation — FAO Framework*

The **[TAPE](https://www.fao.org/agroecology/database/detail/en/c/1158020/)** is an FAO framework that assesses farm and food systems through **10 performance elements** across environmental, social, and economic dimensions, guiding transitions toward sustainable agroecology.

**My Contributions:**

| Component | Description |
|-----------|-------------|
| 📋 **KoboToolbox Survey** | XLSForm development for TAPE data collection |
| 🧮 **Performance Indicators** | Complete R implementation of all TAPE scoring algorithms |
| 📊 **Composite Indices** | FIES, WDDS, A-WEAI, Biodiversity, Soil Health |
| 🔬 **Validation Tools** | Data quality checks and indicator cross-validation |

**Indicators Implemented:**

```r
✅ Food Security (FIES — 8 item Rasch)     ✅ Dietary Diversity (WDDS — 10 food groups)
✅ Soil Health Score (10 parameters)        ✅ Economic Perception Index
✅ Land Tenure (3 components)              ✅ Youth Empowerment Score
✅ Women's Empowerment (A-WEAI + GPI)      ✅ Biodiversity (Gini-Simpson Index)
```
