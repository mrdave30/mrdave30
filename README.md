## 🌱 TAPE Work

> *Tool for Agroecology Performance Evaluation — Food and Agriculture Organization (FAO)*

The **[TAPE](https://www.fao.org/agroecology/database/detail/en/c/1158020/)** is an FAO framework assessing farm and food systems through **10 performance elements** across environmental, social, and economic dimensions, guiding sustainable transitions in agriculture.

**My Contributions:**

| Component | What I Built |
|-----------|-------------|
| 📋 **KoboToolbox Survey Design** | XLSForm development for structured TAPE data collection |
| 🧮 **Performance Indicators (R)** | 1,017-line implementation of all TAPE scoring algorithms |
| 📊 **Composite Indices** | FIES, WDDS, A-WEAI, Biodiversity Index, Soil Health Score |
| 🔬 **Data Validation** | Automated quality checks and cross-indicator validation functions |
| 📈 **Tableau Visualisation** | Interactive TAPE dashboards for IFAD Burundi & Sophie Project |

**Indicators Implemented in R:**

```r
✅ Food Security          FIES — 8-item Rasch model, inverted scoring
✅ Dietary Diversity      WDDS — 10 food groups, sub-item max aggregation
✅ Soil Health            10 parameters scaled 1–5, formula: ((sum/10)−1)×25
✅ Economic Perception    6 items on 0–4 scale, excludes −99 from denominator
✅ Land Tenure            3 components: legal recognition, perception, rights
✅ Youth Empowerment      Individual + aggregate repeat-group scoring
✅ Women's Empowerment    A-WEAI = GPI×0.1 + Women_5DE×0.9×100
✅ Biodiversity           Gini-Simpson Index = 100×(1−Σpi²/area²)
```
